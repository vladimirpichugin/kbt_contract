## Для начала
> **Python 3.7**

Если есть какие-то ошибки при запуске, то первым делом выполнить команду для установки зависимостей
```shell
python -m pip install -r requirements.txt
```
**Пример конфигурации в файле** `settings_example.py`

## Запуск
```shell
python runner.py
```
